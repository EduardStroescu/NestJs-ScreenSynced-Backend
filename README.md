## Description

Recreated current ScreenSynced backend which was using ExpressJs and MongoDb with NestJs, PostgreSql and Prisma. Also included github actions for CI/CD.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
